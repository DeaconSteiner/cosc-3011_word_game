# Contributing to cosc‑3011_word_game

## 🧭 How to Contribute

### 1. Fork the Repository
1. Click **Fork** at the top right of this repo.
2. This creates your own copy under your GitHub account.

### 2. Clone Your Fork Locally
```
git clone https://github.com/YOUR‑USERNAME/cosc‑3011_word_game.git
cd cosc‑3011_word_game
```

### 3. Create a New Branch  
Always use a new branch for each set of changes:
```
git checkout ‑b feature/feature-name
```

### 4. Make Your Changes
Make code improvements, fix bugs, add features, or update documentation.

### 5. Commit & Push
```
git add .
git commit ‑m "Short, clear description of change"
git push origin feature/feature-name
```

### 6. Open a Pull Request
1. Go to your fork on GitHub.
2. Click **Compare & Pull Request**.
3. Provide a descriptive title and summary.
4. Submit the PR to the **main branch** of the original repo.

---

## 🛠 Development Guidelines

- Write clear commit messages.
- Follow existing code style.
- Include comments and documentation for new code.
- Don’t push directly to this repository’s main branch — always use a fork & PR.

---

## 🐛 Reporting Issues

If you find a bug, open an **Issue** and include:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Screenshots or error messages (if helpful)
